# ✊✋✌️ Stone Paper Scissors — Interactive Web Game

A responsive and interactive **Stone Paper Scissors** game built using core web technologies. This project demonstrates fundamental front-end development skills, including DOM manipulation, event handling, game logic implementation, and persistent state management using browser storage.

## 🌐 Live Demo
👉 https://your-username.github.io/stone-paper-scissors-game/

## 📌 Overview

This project is a browser-based implementation of the classic **Stone Paper Scissors** game where the user plays against a computer opponent. The computer generates random moves, and the result is calculated instantly based on standard game rules.

The application also includes persistent scoring and enhanced interactivity features such as auto-play and keyboard controls.

## 🚀 Key Features

- 🎮 Interactive player vs computer gameplay  
- 🤖 Randomized computer move generation  
- 📊 Real-time score tracking (Wins / Losses / Ties)  
- 💾 Persistent data storage using `localStorage`  
- ⚡ Auto Play mode for continuous gameplay  
- ⌨️ Keyboard shortcuts for faster interaction:
  - `R` → Rock  
  - `P` → Paper  
  - `S` → Scissors  
- 🖼️ Visual representation using emoji-based UI  
- 📱 Responsive and lightweight design  

## 🛠️ Tech Stack

- **HTML5** — Structure and layout  
- **CSS3** — Styling and UI design  
- **JavaScript (Vanilla JS)** — Game logic and interactivity  
- **DOM API** — Dynamic UI updates  
- **LocalStorage API** — Persistent score tracking  

## 🧠 Core Concepts Demonstrated

- Event-driven programming  
- Conditional logic and decision-making systems  
- Randomization using `Math.random()`  
- State management in the browser  
- Persistent storage with `localStorage`  
- Modular JavaScript functions

## 🎮 How to Play

1. Open the game in a browser  
2. Choose Rock, Paper, or Scissors  
3. Computer generates its move automatically  
4. Result is displayed instantly  
5. Score updates and is saved automatically
